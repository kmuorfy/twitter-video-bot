# Twitter/X Video İndirici Bot

Telegram üzerinden Twitter/X videolarını indiren bot.

## Kurulum (Railway)

1. Bu klasörü GitHub'a yükle
2. Railway'de "New Project" → "Deploy from GitHub repo" seç
3. BOT_TOKEN environment variable ekle
4. Deploy et!

## Kullanım

- `/start` - Botu başlat
- `/help` - Yardım
- Herhangi bir Twitter/X linki gönder → video gelir!
